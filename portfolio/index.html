Rural IVRS

For: Indian Institute of Technology Bombay

Duration: May 2015 – Present

Description: Rural IVRS is an Interactive Voice Response System(IVRS) targeted to the low end consumers and producers. IVRS generates a voice response that is used to generate bill. Project is guided by Prof. Ganesh Ramakrishnan of CSE Department, IITB. The project follows MVC architecture very closely.

The project is being developed in Spring Framework, Java, Kookoo Telephony Engine, MySQL, Thymeleaf, Bootstrap and AngularJS.

My Role: Contributed in design and development of service layer. Designed a daemon module to download and store files on IITB server regularly.

EpsomUI

For: Persistent Systems Limited

Duration: May 2014 – July 2014

Description: EpsomUI was two months Internship project while working as Project Trainee in Persistent Systems Limited.

EpsomUI detects the changes in the successive tests run on the webpage by comparing screenshots as well as page structure. Thus any change in the UI of the website can be detected easily. Further it also checks the browser compatibility of various elements.

My Role: Developed code for comparing page structure and verifying browser compatibility.
