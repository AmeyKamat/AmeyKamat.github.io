Achievements

Publications

Tanay Gahlot, Amey Kamat and Dr. Pravati Swain, "Automation Toolpath Generation for 3-Axis CNC" in IEEE International Conference on Indusrtial Technology (ICIT), Taipei, Taiwan, Mar. 2016 (in press).
Abstract— In this paper we present toolpath generator which can produce steps of manufacturing along with the toolpaths for any 3D volumetric model. To achieve this automation, we propose sequence generation algorithm that generates sequence of setup orientations required to machine the design. The algorithm treats sequence generation as a search problem and finds shortest sequence of orientations using beam search strategy. Secondly, we suggest an operation planning algorithm that produces near optimal sequence of operations that minimizes no-cut time. Finally, we use this sequence of operations to generate G-code using zig-zag toolpath generation strategy. We tested the toolpath generated by our system against the toolpaths generated by expert CNC machinist using open source CAM softwares for 10 case studies.We found that sequence of orientations and operations generated by our system to be exactly same as the one’s performed by machinist.
